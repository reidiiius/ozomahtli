### Usage
command line

    cd ozomahtli

    js -s rhinestone.js | less

or filter with grep

    js -s rhinestone.js | grep -B2 -A14 'n0'

    js -s rhinestone.js | grep -B2 -A14 'zh' | less

set execute permission

    chmod u+x rhinestone.js

    ./rhinestone.js | less
