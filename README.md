### Usage
command-line

    cd ozomahtli

:

    js rhinestone.js | less

or filter with grep

    js rhinestone.js | grep -B2 -A14 'n0'

:

    js rhinestone.js | grep -B2 -A14 'zh' | less

set execute permission

    chmod u+x rhinestone.js

:

    ./rhinestone.js | less
